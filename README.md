This is the OpenGL game dedicate to Advance Computer Graphic 1 Class

Project focusing on

- Making fighting game with essential features (Ex. type of attacks, frame freeze, attack feedback)
- Making game using OpenGL through LearnOpenGL guideline

Controls

P1

Move - WASD
Punch (Mid Attack) - F
Kick (High/Overhead Attack) - G

P2

Move - Arrow keys
Punch (Mid Attack) - J
Kick (High/Overhead Attack) - K
